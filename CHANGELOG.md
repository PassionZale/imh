# Changelog

## [1.1.3]

### Changed
- 图表 TAB 切换时重新加载数据，优化开发调试体验
- 图表数据范围调整为近 8 个月

## [1.1.2]

### Features
- 首页车辆卡片新增油耗/油费趋势图表（基于 fl_chart）
  - 油耗折线图：展示每次加油数据点，支持触摸查看详情
  - 油费柱状图：按月汇总，支持触摸高亮交互
  - TAB 懒加载切换，数据范围可配置（默认近 12 个月）
- 更新 App 图标

## [1.1.1]

### Bug Fixes
- 修复 release 构建网络不可用的问题

## [1.1.0]

### Features
- 新增拍照识别油耗功能，支持通过 AI 视觉自动提取加油信息（总费用、加油量、日期）
- 新增环境变量设置页，支持配置 LLM 接入信息（OpenAI 兼容格式）

## [1.0.3]

### Bug Fixes
- 修复首页车辆卡片网格单元格溢出
- 修复打卡天数计算忽略频率的 bug
- 修复打卡日历底部间距和 Android debug 构建冲突

## [1.0.1]

### Features
- 重构油耗记录列表页 UI 为时间轴卡片布局
- 新增打卡记录列表页，支持月度统计和打卡日历

### Bug Fixes
- 修正平均油耗计算逻辑

## [1.0.0]

### Features
- 用户创建和个人资料管理
- 每日打卡任务管理（创建、打卡、日历视图）
- 车辆管理和油耗记录（加油量、总费用、里程追踪、油耗统计）
- 数据导入功能（JSON 格式导入油耗和打卡记录）
- 首页展示打卡任务和车辆信息
- Debug 模式屏幕常亮

### Infrastructure
- SQLite 本地数据库
- SharedPreferences 用户会话持久化
- Material Design 主题系统
- 通用 UI 组件库（卡片、空状态、图片预览）
